## Overview

TaskFlow was built as a learning project to explore real-time collaborative features and complex drag-and-drop interfaces. The goal was to create a lightweight alternative to heavy project management tools.

## Key Features

- **Kanban Boards**: Drag-and-drop task management with customizable columns
- **Real-time Updates**: Live collaboration via WebSocket connections
- **Team Workspaces**: Create and manage team spaces with invite links
- **Role-based Access**: Admin, editor, and viewer permission levels
- **Search & Filter**: Full-text search across tasks with advanced filtering

## Technical Highlights

Next.js provides server-side rendering for fast initial loads, while client-side hydration enables the rich drag-and-drop interface. Prisma handles database migrations and type-safe queries against PostgreSQL. The WebSocket server broadcasts changes to all connected team members in real-time.
