## Overview

As a former top 150 Counter-Strike player in the Netherlands, I built this tool to analyze my gameplay patterns and identify areas for improvement. The dashboard pulls real-time data from multiple gaming APIs and presents actionable insights.

## Key Features

- **Live Stats**: Real-time ELO, win rate, and K/D ratio tracking
- **Match History**: Detailed match breakdowns with scoreboard data
- **Performance Trends**: Visualize improvement or decline over customizable time periods
- **Map Analysis**: Win rates and performance metrics per map
- **Comparison Tool**: Compare your stats against players at similar or different ranks

## Technical Details

The FastAPI backend handles API aggregation and data normalization from Steam and FACEIT. PostgreSQL stores historical match data for trend analysis, while Redis provides a caching layer to stay within API rate limits. The React frontend features responsive charts and real-time data updates.
