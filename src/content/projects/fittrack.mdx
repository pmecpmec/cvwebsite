## Overview

Fittrack was born from my personal need to track gym workouts more effectively than existing apps allowed. The application supports detailed exercise logging, volume tracking, and progressive overload analysis.

## Key Features

- **Workout Logger**: Log exercises, sets, reps, and weights with an intuitive interface
- **Progress Charts**: Interactive Chart.js visualizations showing strength progression over time
- **Training Plans**: Template-based workout plans with customizable split routines
- **Body Metrics**: Track weight, measurements, and body composition changes
- **REST API**: Full RESTful API with JWT authentication

## Technical Architecture

The frontend uses React with custom hooks for state management, while the backend runs on Node.js with Express. MongoDB was chosen for its flexibility in modeling varied exercise data structures. The API includes rate limiting, input validation, and comprehensive error handling.
