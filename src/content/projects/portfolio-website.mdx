## Overview

This portfolio website represents a complete redesign and migration from a static HTML/CSS/JS site to a modern Astro-based architecture. The goal was to create a fast, accessible, and visually polished showcase while maintaining excellent performance.

## Key Features

- **Partial Hydration**: JavaScript only loads where needed, keeping the runtime minimal
- **Progressive Enhancement**: Fully functional without JavaScript enabled
- **Dark/Light Mode**: Smooth theme transitions with system preference detection
- **Responsive Design**: Optimized layouts for all screen sizes
- **SEO Optimized**: OpenGraph, Twitter Cards, JSON-LD structured data
- **Content Management**: MDX-based project content for easy updates

## Technical Highlights

The architecture leverages Astro's island architecture to deliver primarily static HTML with interactive Svelte components hydrated only when visible in the viewport. This approach results in near-zero JavaScript for static content while providing rich interactions where needed.
